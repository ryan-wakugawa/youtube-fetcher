# Trabalho Prático de NoSQL

Utilizem a biblioteca youtool para acessar a API do YouTube e coletar dados como:

- Comentários
- Transcrições
- Livechats
- Superchats

1. Escolham um canal do YouTube como objeto de análise.
2. Utilizem o MongoDB (ou outro banco de dados NoSQL) para armazenar e trabalhar os dados coletados.
